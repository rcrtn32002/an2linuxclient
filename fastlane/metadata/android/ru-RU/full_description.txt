AN2Linux позволяет синхронизовать уведомления с вашего телефона на компьютер с Linux и шифровать их при помощи TLS.

Это можно делать через WiFi, мобильную сеть или Bluetooth.

Для работы AN2Linux на компьютере должен быть запущен AN2Linux-server.

Для получения дополнительной информации смотрите файл README в git-репозитории.
Если вы столкнулись с какой-то проблемой, то создайте Issue на GitHub.

AN2Linux это свободное ПО с открытым исходным кодом, выпущенное под лицензией GNU GPL v3.

Исходный код AN2Linux-server:
